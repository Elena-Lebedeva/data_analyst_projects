# Data Analytics & Exploratory Data Analysis (EDA)

This repository contains a collection of end-to-end data science projects and technical case studies developed during a professional certification in Data Analytics by **Practicum by Yandex** (250+ hours of audited code reviews).

These projects demonstrate my proficiency in using **Python** and its analytical ecosystem to extract actionable insights from raw, unstructured datasets to drive business decisions.

## 📂 Repository Highlights

### 1. Business Metrics & Funnel Analysis
* **Objective:** Evaluating core business KPIs, conversion funnels, and customer retention.
* **Technical Focus:** Cohort analysis, LTV (Lifetime Value) calculation, and identifying friction points in the user journey.
* **Tools:** Pandas, Seaborn.

### 2. Hypothesis Testing & A/B Analytics
* **Objective:** Applying statistical methods to validate business decisions and experimental design.
* **Technical Focus:** Setting up A/B test frameworks, calculating p-values, and ensuring statistical significance.
* **Tools:** SciPy, Statsmodels.

### 3. Exploratory Data Analysis (EDA) Case Studies
* **Objective:** Transforming raw datasets into structured narratives and clean data models.
* **Technical Focus:** Data cleaning (handling missing values/outliers), feature engineering, and correlation analysis.
* **Tools:** Matplotlib, NumPy.

---

## 🛠 Technical Stack & Training
* **Language:** Python 3.x (Pandas, NumPy, Matplotlib, Seaborn, SciPy, Statsmodels)
* **Databases:** SQL
* **Environment:** Jupyter Notebooks, Google Colab
* **Curriculum:** Data Preprocessing, EDA, Statistical Analysis, Marketing Cost Optimization, and Basic Machine Learning (Classification, Clustering, Regression).

---

## 🔬 Methodology: The Sociological Approach
In each of these projects, I apply a **Sociological Research Framework**. I don't just look at the "what" in the numbers, but the "why" behind the human behavior. This dual-lens approach allows for deeper root-cause analysis regarding customer behavior and service journey integrity.
