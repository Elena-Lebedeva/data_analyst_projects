# Data Analytics & Exploratory Data Analysis (EDA)

This repository contains a collection of end-to-end data science projects and technical case studies. These projects demonstrate my proficiency in using **Python** and its analytical ecosystem (**Pandas, NumPy, Matplotlib, Seaborn**) to extract actionable insights from raw, unstructured datasets.

## 📂 Repository Highlights

### 1. Business Metrics & Funnel Analysis
* **Objective:** Evaluating core business KPIs, conversion funnels, and customer retention.
* **Technical Focus:** Cohort analysis, LTV (Lifetime Value) calculation, and identifying friction points in the user journey.
* **Tools:** Pandas, Seaborn.

### 2. Hypothesis Testing & A/B Analytics
* **Objective:** Applying statistical methods to validate business decisions.
* **Technical Focus:** Setting up A/B test frameworks, calculating p-values, and ensuring statistical significance in operational changes.
* **Tools:** SciPy, Statsmodels.

### 3. Exploratory Data Analysis (EDA) Case Studies
* **Objective:** Transforming raw datasets into structured narratives.
* **Technical Focus:** Data cleaning (handling missing values, outliers), feature engineering, and multi-variable correlation analysis.
* **Tools:** Matplotlib, NumPy.

---

## 🛠 Technical Stack
* **Language:** Python 3.x
* **Libraries:** * **Data Manipulation:** Pandas, NumPy
    * **Visualization:** Matplotlib, Seaborn
    * **Statistics:** SciPy, Statsmodels
* **Environment:** Jupyter Notebooks / Google Colab

---

## 🔬 Methodology: The Sociological Approach
In each of these projects, I apply a **Sociological Research Framework**. This means I don't just look at the "what" in the numbers, but the "why" behind the human behavior. This dual-lens approach allows for deeper root-cause analysis and more accurate predictive insights regarding customer dissatisfaction (DSAT) and service journey integrity.
